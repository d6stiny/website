---
title: O que é C++?
date: 11/16/2023
---

C++ é uma linguagem de programação de alto nível, compilada e multi-paradigma (programação funcional, programação orientada a objetos, programação genérica, programação procedural e programação imperativa), criada em 1979 por Bjarne Stroustrup na Bell Labs como uma extensão do C.

## Aplicações do C++

Pode ser utilizado para fazer:

- Desktop
- Mobile
- Embedded Applications
- Jogos
- Sistemas Operacionais
- Compiladores e Interpretadores de outras linguagens

## Exemplo de código em C++

```cpp
#include <iostream>

int main() {
    std::cout << "Hello World!";
    return 0;
}
```
