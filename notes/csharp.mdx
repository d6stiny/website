title: O que é C#?
date: 11/16/2023
---

C# é uma linguagem de programação de alto nível, compilada, multi-paradigma, tipada, de propósito geral. Foi criada em 2000 por Anders Hejlsberg na Microsoft.

## Aplicações do C#

Pode ser utilizado para fazer:

- Desktop
- Mobile
- Embedded Applications
- Jogos

## Exemplo de código em C#

```cs
using System;

namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello World!");
        }
    }
}
```
