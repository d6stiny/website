Bases de dados (databases) são coleções de dados que são organizados de forma a facilitar a sua utilização. São utilizadas para armazenar dados de forma estruturada, com a finalidade de facilitar a sua consulta e pesquisa.

# Tipos de bases de dado

Existem vários tipos de bases de dados, sendo que as mais comuns são:

- Relacionais (SQL)
- Não relacionais (NoSQL)
- Orientadas a objetos (OODBMS)
- Hierárquicas (Hierarchical)
- Em rede (Network)

## Microsoft Access

Microsoft Access é um sistema de gerenciamento de banco de dados da Microsoft, incluído no pacote do Microsoft Office Professional, que combina o Microsoft Jet Database Engine com uma interface gráfica do utilizador. Este programa permite a criação e manutenção de bases de dados, ou seja, coleções de dados estruturadas, relacionadas entre si, e que podem ser utilizadas por várias aplicações.

## MySQL

MySQL é uma base de dados gratuita, de código aberto, muito popular, que utiliza a linguagem SQL (Structured Query Language). O MySQL é muito utilizado em aplicações web, como o WordPress, Drupal, Joomla, etc.

## MongoDB

MongoDB é uma base de dados NoSQL, orientada a documentos, de código aberto, que utiliza documentos semelhantes a JSON com esquemas.
