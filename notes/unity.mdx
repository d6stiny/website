## O que é o Unity?

O Unity é uma das maiores engines de jogos. É usado para criar jogos, tanto 2D como 3D para qualquer plataforma, Linux, Mac, Windows, iOS, Android e VR/AR. Oferece um editor gráfico e ferramentas para preencher os requisitos de qualquer jogo.

Além disso, o Unity tem uma das maiores comunidades de desenvolvedores, possuindo assim uma das maiores Assets Store.

## Diferenças entre 2D e 3D

2D e 3D são dimensões de objetos e imagens.

2D têm apenas 2 dimensões, X e Y, enquanto o 3D tem 3 dimensões, X, Y e Z.

3D é mais usado para tornar as coisas mais imersivas pois é a dimensão que os seres humanos usam.

## Quando surgiu o Unity?

O Unity foi anunciado primeiramente na WWDC 2005 da Apple, a versão inicial era limitada a Mac&apos;s e foi lançada a 8 de Junho de 2005.

## Aplicações do Unity

O Unity pode ser usado para jogos e aplicações de qualquer tamanho. Jogos 2D ou 3D. Aplicações de AR e aplicações desktop, web ou mobile. Filmes e animações.

## Linguagens que podemos usar no Unity

Unity suporta, por padrão, apenas C# e é depois compilado para a plataforma definida.
