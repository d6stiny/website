date: 11/07/2023
---

**HTML** (Hyper-Text Markup Language) é uma linguagem de **marcação** (não confundir com linguagem de programação) utilizada para estruturar páginas web. Foi lançado em 1993 por Tim Berners-Lee e é utilizado até hoje para estruturar páginas web.

## Estrutura básica

```html
<!DOCTYPE html>
<html lang="pt-pt">
  <head>
    <meta charset="UTF-8" />
    <title>Title</title>
  </head>
  <body>
    <h1>Olá Mundo!</h1>
  </body>
</html>
```

# Estilos

Para adicionar estilos a uma página web, podemos utilizar [CSS](./css) (Cascading Style Sheets).
